# Install

LuneWeb has an npm package which contains libraries
for interacting with the internal functions that are
inserted into the global `window` object

::: code-group

```shell [yarn]
yarn add luneweb
```

```shell [npm]
npm add luneweb
```

```shell [pnpm]
pnpm add luneweb
```

```shell [bun]
bun add luneweb
```

:::
