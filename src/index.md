# Install

## Recommended methods

::: code-group

```shell [rokit]
rokit init
rokit add luneweb/luneweb
```

```shell [cargo-install]
cargo install --git https://github.com/luneweb/luneweb
```

:::

## Alternatives

- aftman
- foreman

## Install manually

You can manually download LuneWeb from
[Github](https://github.com/luneweb/luneweb/releases/latest)
